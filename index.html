HTML

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat konten untuk World Wide Web (WWW). HTML terdiri dari serangkaian tag dan atribut yang digunakan untuk mendefinisikan struktur dan tampilan halaman web. Dokumen HTML diinterpretasikan oleh web browser untuk menampilkan halaman web kepada pengguna.
